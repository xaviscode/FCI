# Financial Crisis Index for Systemic Risk Monitoring

This project develops a reproducible framework to monitor systemic financial stress across four pilot countries: the United States, Germany, Spain, and the United Kingdom.

## Objective

The main objective is to build a Financial Crisis Index (FCI) using open macro-financial, market, leverage, and uncertainty indicators. The project combines data engineering, index construction, validation, and cross-country interpretation into a single workflow.

## Project structure

The project is organized into six work packages:

- **WP1** — Project definition, conceptual framework, indicator selection, and planning
- **WP2** — Data acquisition, harmonisation, and storage pipeline
- **WP3** — Construction of baseline indices:
  - Equal-weight Financial Crisis Index
  - PCA-based Financial Crisis Index
- **WP4** — Validation using historical stress windows, episode summaries, and peak analysis
- **WP5** — Cross-country discussion and interpretation of the results
- **WP6** — Final conclusions, limitations, and future work

## Main outputs

The project produces:

- A harmonised multi-country macro-financial panel.
- Block sub-indices for market stress, credit & leverage, macro vulnerability, and uncertainty.
- Two baseline Financial Crisis Indices.
- Validation tables and figures.
- Cross-country analytical summaries.

## Key contribution

The project provides a transparent and reproducible systemic-risk monitoring framework that combines:

- Interpretable indicator blocks.
- Reproducible ETL pipelines.
- Complementary index constructions.
- Country-level empirical analysis.

## Reproducibility

The full workflow is version-controlled and designed to be reproducible, from raw data acquisition to final analytical outputs.
